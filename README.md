# IonianFarsi - Polyphonic Application  

## Abstract  
IonianFarsi is a multimedia online extracurricular application designed for the learning and reinforcement of the Persian language. Utilizing the 1stStep book layout—words, lessons (17 lessons), and book images—the application will implement sections “Lessons” and “Exercises.” For audio files, speechgen.io will be utilized.

---

## Features  
- **Lessons**: Interactive lessons following the 1stStep book structure.  
- **Exercises**: Practice exercises for vocabulary reinforcement.  
- **Audio Integration**: Speech synthesis using speechgen.io.  
- **Database**: MySQL database for managing course-related words.  
- **API**: Node/Express.js API for seamless interaction with the database.  
- **Front-End**: Developed with React.js for a dynamic user interface.

---

## Methodology  
- **Database Setup**: Create a MySQL database with course-related words from the 1stStep book.  
- **API Development**: Develop an API for the application's interaction with the database using Node/Express.js.  
- **Front-End Development**: Build the application's user interface using React.js.  
- **Prototyping Model**: The application will follow a prototyping approach during development.
